Welcome, thank you for visiting.

My goal for this project was to take all of the wonderful lists and tables created by others and myself to create automatic tools that make using the printed or digital materials simple, modular, and inspiring.

I did not create the majority of the content you see on this site, simply adapt it to a new format from the printed/ manual generation methods I found or bought them in.

I hope these tools are useful to you.
